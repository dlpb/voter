Simple mouse voter
==================

A java implementation of the mouse voter. Multiplexes for a three button mouse to give five inputs in total. 

Requires JNativeHook 1.1.4 - not included in this repository

Buttons
=======

Button Number -> Name
1 -> Left
2 -> Right
3 -> Middle

Vote -> Button Combination
1 -> 2
2 -> 2 | 1
3 -> 3
4 -> 3 | 1
5 -> 3 | 2
